HfdScroll

This is a custom scroller I implemented for an old version of hungryfordesign.com. It has a few interesting features:

  • The scrollbar can be thrown and will bounce off the edges of the container if thrown hard enough.
  • The buttons with the arrows on them can be pulled "out" of the scrollbar.
